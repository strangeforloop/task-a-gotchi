# Universal clock-time badge on every task row

## Key Decisions

- Every task displays a clock-time badge (e.g. "8:00 AM", "3:45 PM") in the title row
- Habits with scheduledTime → use that time
- Habits without scheduledTime → "8:00 AM" (start-of-day default)
- Template tasks → "8:00 AM" (day-start convention)
- One-off tasks → wall-clock time derived from their createdAt unix ms timestamp
- Badge color matches the source dot (cyan=habit, purple=template, green=one-off)
- Remove "added X ago" relative label — clock time badge supersedes it

## File / Component Structure

```
src/utils/format.ts                  — add formatTaskTime(source, createdAt?, habitScheduledTime?)
src/components/tasks/TaskRow.tsx     — compute timeDisplay, render universal badge, remove addedLabel
```
