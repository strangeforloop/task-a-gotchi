# Long-press "+" FAB to add a recurring task

## Key Decisions

- Tap "+" → one-off task (existing behaviour, green dot)
- Long-press "+" → recurring task via addTemplate (purple dot, haptic feedback)
- Visual mode indicator in open input: dot color + label ("one-off · today" vs "recurring · every Wednesday")
- isRecurring state lives in AddBar, resets after each submit or close
- AddBar gets two new optional props: onAddRecurring and recurringDayLabel
- DAY_LONGS mapping computed in app/index.tsx, passed as recurringDayLabel prop

## File / Component Structure

```
src/components/shared/AddBar.tsx   — add isRecurring state, onLongPress on FAB, mode-aware dot + label + submit
app/index.tsx                      — destructure addTemplate + todayId, compute recurringDayLabel, pass new props
```
