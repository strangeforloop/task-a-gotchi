# Increase habit penalty scales so missed habits visibly impact HP

## Key Decisions

- One missed daily habit at 6–12h overdue should push the pet out of "thriving" (< 90 HP)
- Overdue habit scale raised: 0 / 15 / 25 HP (was 0 / 5 / 10)
- Same-day late habit scale raised: 0 / 5 / 10 HP (was 0 / 3 / 5)
- Late habit scale stays softer than overdue since the habit can still be completed today
- Regular weekly tasks stay at 0 / 10 / 20 / 30 HP — habits remain lighter but meaningfully penalised

## File / Component Structure

```
src/utils/habits.ts   — habitPointsForHours: return values changed to 0/15/25
src/utils/format.ts   — computeHabitLatePoints: return values changed to 0/5/10
```
