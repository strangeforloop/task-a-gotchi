# Late Habit HP Penalty

**Date:** 2026-05-27

## What
Habits with a `scheduledTime` that haven't been completed past their scheduled time should reduce HP, not just show a cosmetic label.

## Penalty scale
Mirrors the existing `habitLateLabel` tiers:
- < 10 min past scheduled time → 0 HP (grace period)
- 10–59 min late → −3 HP  (shown as "Xm late")
- 60+ min late → −5 HP    (shown as "Xh late")

Penalty clears immediately when the habit is completed.

## Files changed

- `src/utils/format.ts` — added `computeHabitLatePoints(hhmm, now): number`
- `src/context/HabitContext.tsx` — added `habitLatePenalty: number` (sum over all scheduled-but-late-incomplete habits today), exposed from context
- `src/hooks/useHealth.ts` — added `latePenalty = 0` param: `Math.max(0, Math.min(100, computeHealth(tasks) + habitBonus - latePenalty))`
- `app/index.tsx` — destructures `habitLatePenalty` from `useHabits()`, passes to `useHealth(allTasksForHp, habitBonus, habitLatePenalty)`
