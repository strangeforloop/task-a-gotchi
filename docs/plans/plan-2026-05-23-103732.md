# Pet / Feed / Talk Buttons + Animations + Tests

## Key Decisions

- Button labels renamed: A → "pet" (affectionate hearts), B → "feed" (cookie pop), C → "talk" (quote)
- Animations use React Native's built-in `Animated` API (not Reanimated) — simpler, no worklet setup needed
- Hearts: 5 staggered `♥` glyphs float upward and fade inside the LCD overlay
- Cookie: single `🍪` springs to full scale then fades out, centered in the LCD
- MessageToast repositioned into a fixed-height layout slot (not absolute) so it never overlaps device or task list
- Tests use `jest-expo` preset + `@testing-library/react-native` for hook testing

## File / Component Structure

```
src/constants/colors.ts         button subs: 'pet', 'feed', 'talk'
src/components/effects/
  HeartsBurst.tsx               5 animated ♥ glyphs floating up/fading (Animated API)
  FoodBowl.tsx                  🍪 spring-pop then fade (Animated API)
  MessageToast.tsx              plain flexbox bubble, no absolute positioning
app/index.tsx                   fixed-height speechSlot between device and task list
jest.config.js                  preset: jest-expo
src/__tests__/
  messages.test.ts              pickMessage() returns string from MOTIVATIONAL_MESSAGES
  useEffectTimer.test.ts        playEffect sets/clears effect; replaces active effect
  buttonLabels.test.ts          BUTTON_DEFINITIONS subs === ['pet', 'feed', 'talk']
```
